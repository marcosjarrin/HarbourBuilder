hbcpp - What's New
==================

2026-03-31: macOS Cocoa/AppKit backend
--------------------------------------
- New backend: backends/cocoa/ with full Cocoa/AppKit implementation
  * cocoa_core.m replaces the Win32 C++ core for macOS
    - TControl, TForm, TLabel, TEdit, TButton, TCheckBox, TComboBox, TGroupBox
    - Flipped NSView for top-left origin coordinate system (Win32 compatible)
    - Design mode with drag, resize, rubber band selection
    - NSApplication initialization with menu bar
    - JSON serialization of form layouts
  * cocoa_inspector.m: property inspector window using NSTableView
    - Category headers with collapse/expand
    - Live property editing with color and font pickers
    - Exports: INS_Create, INS_RefreshWithData, INS_BringToFront, INS_Destroy

- New Harbour files for macOS
  * harbour/inspector_mac.prg: Harbour wrapper for Cocoa inspector
  * samples/test_design_mac.prg: design-mode sample (same UI as Win32 version)
  * samples/build_mac.sh: build script (clang + Harbour + Cocoa framework)

- The existing Harbour OOP layer (classes.prg, commands.ch) works unchanged
  across Win32 and macOS backends
