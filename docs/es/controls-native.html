Sintaxis xBase

// Definir ListView con 3 columnas
@ 50, 20 LISTVIEW oLV OF oForm SIZE 480, 250 ;
   COLUMNS "Nombre", "Edad", "Ciudad"

oLV:AddItem( { "Alice",   "30", "New York"    } )
oLV:AddItem( { "Bob",     "25", "Los Angeles" } )
oLV:AddItem( { "Charlie", "40", "Chicago"     } )

Editar en el Inspector

Al seleccionar un control TListView en el diseñador, el inspector expone dos propiedades editables:

  • aColumns — pulsar el boton ... abre el editor de array. Una columna por linea.
  • aItems — pulsar el boton ... abre el ListView Items Editor: una rejilla con las cabeceras de aColumns, una fila por item, doble click sobre cualquier celda para editarla en sitio. Botones + Add / - Del / Up / Down para gestionar filas. Enter confirma, Esc cancela.

El generador de codigo emite el comando @ ... LISTVIEW ... COLUMNS ... seguido de un :AddItem({...}) por cada fila.

Formato de almacenamiento

Internamente las propiedades aColumns y aItems se serializan como cadenas:

  • aColumns: cabeceras separadas por | — ej. "Nombre|Edad|Ciudad"
  • aItems: filas separadas por |, celdas dentro de cada fila por ; — ej. "Alice;30;NY|Bob;25;LA"

Modos de visualizacion

La propiedad nViewStyle aparece en el inspector como dropdown con los valores enumerados (vsIcon, vsList, vsReport, vsSmallIcon). Cambiar el valor en runtime refresca el control inmediatamente. Los modos Icon y SmallIcon requieren un ImageList asociado para mostrar iconos.

Editar imagenes en el Inspector

La propiedad aImages abre un dialogo dedicado ListView Images Editor al pulsar ...: lista de rutas + botones + Add (file picker .png/.ico/.bmp), - Del, Up, Down. Las imagenes se cargan en runtime via GDI+, escaladas a 32x32 (vsIcon) y 16x16 (vsSmallIcon). Asignacion por fila es round-robin: row[N]:icono = aImages[N mod len(aImages)].

// Definir ListView con iconos
@ 50, 20 LISTVIEW oLV OF oForm SIZE 480, 250 ;
   COLUMNS "Nombre", "Edad" ;
   ITEMS "Alice;30", "Bob;25" ;
   IMAGES "user.png", "admin.png"

oLV:nViewStyle := 0  // Modo Icon: muestra PNGs 32x32
